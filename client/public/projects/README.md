# Project Images

Project assets are organized into subfolders:

```
projects/
├── bvc/           # Barrett Values Centre
│   ├── bvc-logo.svg
│   ├── bvc-dashboard.png
│   ├── bvc-assessments.png
│   └── ...
├── gamehub/       # GameHub
│   ├── gamehub-light-home.png
│   ├── gamehub-dark-games.png
│   ├── gamehub-casual.png
│   └── ...
└── connectxpert/  # ConnectXpert
    └── connect-xpert.png
```

## Image Tips
- Use JPG for photos/screenshots, PNG for graphics with transparency
- Optimize images before uploading (TinyPNG, ImageOptim)
- Keep file sizes under 500KB for faster loading
