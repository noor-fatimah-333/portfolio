# Project Images

Place your project screenshots and images here:

1. **BVC Screenshots**: Save as `bvc-screenshots.jpg` (or `.png`)
   - You can use multiple screenshots combined into one image, or use a collage
   - Recommended size: 1200x800px or similar aspect ratio

2. **GameHub Screenshots**: Save as `gamehub-screenshots.jpg` (or `.png`)
   - Include key screenshots showing the main features
   - Recommended size: 1200x800px or similar aspect ratio

3. **ConnectXpert Poster**: Save as `connectxpert-poster.jpg` (or `.png`)
   - Your poster image
   - Recommended size: 1200x800px or similar aspect ratio

## Image Tips:
- Use JPG for photos/screenshots, PNG for graphics with transparency
- Optimize images before uploading (use tools like TinyPNG or ImageOptim)
- Keep file sizes under 500KB for faster loading
- Maintain consistent aspect ratios for better visual consistency
